# 🌟 NovaBlog

[🇮🇷 مطالعه به فارسی](./README-fa.md)

NovaBlog is a simple and clean blogging platform built with the Laravel framework.

---

## ✨ Features

- Create, edit, and delete blog posts
- Categorize posts
- Add and manage comments
- User authentication and roles
- Simple and clean UI
